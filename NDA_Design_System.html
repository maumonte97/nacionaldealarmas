06 — Efectos ✓ Pilar visual
Glow + Glassmorphism

Las luces ambientales difusas (glow) son el pilar visual del sitio. Definen la atmósfera premium y dan razón de ser a ambos colores de marca.

Glow amarillo
Para CTAs, badges activos, hero highlights, hover states. Atrae la atención hacia acciones.
color: #F8EB01
opacity: 0.15–0.30
filter: blur(40–80px)
Glow azul
Para atmósfera de fondo, profundidad, bordes de sección. Crea la sensación de espacio y tecnología.
color: #0569B3
opacity: 0.10–0.25
filter: blur(60–120px)
Glass surface
Cards, navbar, modals. Transparencia + blur que deja ver los glows debajo. Crea capas de profundidad.
bg: rgba(255,255,255, 0.04–0.08)
border: rgba(255,255,255, 0.08–0.15)
backdrop-filter: blur(12–20px)

09 — Movimiento
Animaciones

Inspiradas en Verisure. Todas las animaciones usan Framer Motion + Lenis smooth scroll.

Animación Trigger Duración Easing Uso
Fade Up Scroll into view 0.6s ease-out Títulos, párrafos, cards
Stagger Children Scroll into view 0.4s + delay 0.1s ease-out Grids de cards, stats, features
Scale In Scroll into view 0.5s spring(1, 80, 10) Iconos, badges, imágenes hero
Parallax Scroll position Continuo linear Fondos de hero, imágenes decorativas
Counter Up Scroll into view 2s ease-in-out Stats (30+, 24/7, 5/5)
Smooth Scroll Always — Lenis default Scroll global del sitio
Hover Lift Mouse hover 0.2s ease Cards, botones
Nav Blur Scroll > 100px 0.3s ease Navbar de transparente a glassmorphism
